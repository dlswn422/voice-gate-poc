# 차단봉/차단기 물리 고장(이상 동작)

## 증상
- "차단봉이 움직이다 멈춰요", "덜컥거려요", "소리가 나요", "올라가다 내려와요", "아예 반응이 없어요"

## 조치 순서(우선순위)
1) 안전 안내: 차단봉 주변 접근 금지, 차량은 정지 상태 유지
2) 반복 동작 시도 금지(모터/기어 손상 위험)
3) 관리자 호출 안내(출구/입구 호출 버튼 또는 안내번호)
4) 필요 시 우회 출구/수동 처리 안내(현장 정책에 따름)

## 추가 질문(필요 시 1개만)
- "차단봉이 '움직이다 멈춤'인가요, 아니면 '아예 반응 없음'인가요?"

## 에스컬레이션
- 현장 설비 점검 필요(모터/센서/제어기) → 관리자 즉시 조치
