# 할인/무료시간 미적용 (할인등록/무료주차/쿠폰) - DISCOUNT_FREE_TIME_ISSUE

## 대표 키워드
- 할인 안 들어감
- 무료시간 미적용
- 쿠폰
- 영수증
- 매장 등록
- 방문처 등록
- 정산
- 재정산
- 요금 그대로
## 필수 슬롯 (required_slots)
- symptom: 어떤 할인/무료시간 문제인지(할인 미적용/쿠폰 적용 실패/요금 불일치 등)
- where: 어디에서 확인했는지(키오스크/정산기/모바일)
## 추가 슬롯 (optional_slots)
- attempted: 매장/안내데스크에서 등록을 받았는지, 재조회/재정산을 해봤는지
- when: 할인 등록 시점(결제 전/후),
- error_message: 화면 메시지(쿠폰 오류, 적용 불가 등)
## 첫 질문 (ASK_1)
할인이 어떤 방식으로 적용되어야 하나요? (매장 등록/쿠폰/무료시간 중 어떤 건가요?)
## 슬롯 추출 가이드
- '쿠폰 썼는데 그대로예요' → symptom=쿠폰 적용 실패
- '무료시간 안 됐어요' → symptom=무료시간 미적용
- '할인 등록했는데 안 들어가요' → symptom=매장 등록 미적용
- where: '정산기/키오스크' 언급 → 키오스크, '앱/모바일' 언급 → 모바일
- attempted: '등록 받았어요/안 받았어요/영수증 있어요' 등 확인
## 진단 분기 (결정 규칙)
1) 매장 등록이 필요한 유형인데 등록이 안 됐으면 → 매장/안내데스크에서 차량번호로 등록 안내
2) 등록 후에도 미적용이면 → 키오스크에서 차량번호 확인 후 재조회/재정산 안내
3) 중복 할인 제한/조건 미충족(유효시간/최소 결제액 등) 가능성 안내
## 조치 순서 (우선순위)
1) 할인 유형(매장등록/쿠폰/무료시간)과 적용 조건(영수증/방문 확인)을 먼저 확인
2) 매장/안내데스크에서 차량번호로 할인 등록이 필요한 경우 해당 위치로 안내
3) 등록 후 키오스크에서 차량번호 확인 → 요금 재조회/재정산
4) 여러 할인 중복 제한이 있을 수 있음을 안내
5) 그래도 불일치면 관리자/관제에 확인 요청
## 해결 안내 문장 템플릿 (SOLVE_TEMPLATE)
- 할인/무료시간은 방문처에서 차량번호로 등록되어야 적용되는 경우가 많아요. 매장/안내데스크에서 등록 여부를 확인해 주세요.
- 등록 후에는 키오스크에서 차량번호를 확인하고 요금을 다시 조회(재정산)해 주세요.
- 그래도 요금이 불일치하면 관리자 확인이 필요할 수 있어요. 다른 문제가 더 있으신가요?
## 에스컬레이션 조건
- 등록이 정상으로 보이는데도 요금이 계속 동일함
- 여러 단말에서 동일하게 미적용 발생
- 시스템 오류 문구가 표시됨(쿠폰 오류/서버 오류 등)
