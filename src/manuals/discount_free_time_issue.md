# 할인/무료시간 적용 안 됨

## 증상
- "무료시간인데 요금이 나와요", "할인이 적용이 안돼요", "등록했는데도 요금이 떠요"

## 조치 순서(우선순위)
1) 할인/무료 적용 조건 확인(방문등록/제휴할인/정기권/무료쿠폰 등)
2) 적용이 '정산 화면'에 반영됐는지 확인 후, 반영 안 됐다면 재시도 안내
3) 방문등록 기반 할인이라면: 방문등록 완료 여부/차량번호 일치 여부 확인
4) 계속 미반영 시: 현장 단말/서버 반영 지연 가능 → 관리자 확인 요청
5) 출차가 급하면 출구 호출 버튼/안내번호로 도움 요청

## 추가 질문(필요 시 1개만)
- "어떤 할인(방문등록/정기권/쿠폰) 적용을 기대하신 건가요?"

## 에스컬레이션
- 차번호 불일치/등록 누락/시스템 지연 가능 → 관리자 확인 요청
