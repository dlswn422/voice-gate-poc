# 입구 차단기 안 열림 (입차 실패/입구 차단봉 미상승) - ENTRY_FLOW_ISSUE

## 대표 키워드
- 입구 차단기 안 열림
- 입차 안 돼요
- 정기권
- 티켓
- 번호판 인식
- LPR
- 반응 없음
## 필수 슬롯 (required_slots)
- symptom: 어떤 현상인지(차단기 미동작/티켓 태그 무반응/번호판 인식 실패 등)
- where: 입구 차단기/차량 인식 구간 중 어디인지
## 추가 슬롯 (optional_slots)
- error_message: 입구 단말/화면 오류 문구
- attempted: 앞뒤 이동/재태그/다른 차로 시도 여부
## 첫 질문 (ASK_1)
입차에서 어떤 현상이 문제인가요? (차단기 자체가 안 열림/티켓·정기권 태그 무반응/번호판 인식 실패 중 어떤 건가요?)
## 슬롯 추출 가이드
- '티켓 찍어도 반응없' → symptom=태그 무반응
- '번호판이 인식이 안돼' → symptom=번호판 인식 실패
- '차단봉이 안 올라가' → symptom=차단기 미동작
## 조치 순서 (우선순위)
1) 정지선/루프코일 위에 정확히 정차(앞뒤 0.5m 이동 후 재감지)
2) 입차 방식 확인: 티켓/정기권/번호판 인식(LPR) 중 무엇인지 확인 후 재시도
3) 티켓/정기권이면 카드 방향/인식부 위치 확인 후 재태깅
4) 번호판 인식(LPR)이면 번호판 오염/가림 여부 확인 후 닦고 재시도
5) 계속 실패하면 다른 차로 이용 또는 관리자 호출
## 해결 안내 문장 템플릿 (SOLVE_TEMPLATE)
- 차량을 정지선(루프코일) 위에 맞추고 앞뒤로 조금 이동한 뒤 다시 시도해 주세요.
- 티켓/정기권이면 카드 방향과 인식부 위치를 맞춰 재태깅해 주세요.
- 번호판 인식 문제면 번호판을 닦고 다시 인식되도록 시도해 주세요.
- 다른 문제가 더 있으신가요?
## 에스컬레이션 조건
- 동일 차로에서 반복 실패 + 화면 오류 문구 발생
- 여러 차량에서 동시에 발생(장비/네트워크 장애 가능)
