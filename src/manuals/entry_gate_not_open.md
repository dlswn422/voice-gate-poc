# 입차가 안 됨(입구 차단기 안 열림)

## 증상
- "입차가 안돼요", "입구 차단기가 안 올라가요", "차단봉이 반응이 없어요(입구)"

## 조치 순서(우선순위)
1) 차량 위치 확인: 차단기/센서 앞 정위치(너무 멀거나 너무 가까우면 인식 실패)
2) 전광판/화면에 안내 문구 확인(인식 실패/정원 초과/장애 등)
3) 차량 번호 인식 기반이면: 번호판 가림/오염 여부 확인
4) 계속 실패하면: 입구 호출 버튼/안내번호로 관리자 호출 안내
5) 안전: 무리하게 진입/후진 반복하지 않도록 안내

## 추가 질문(필요 시 1개만)
- "입구 화면에 안내 문구가 뜨나요? (예: 인식 실패/잠시만 기다려 주세요)"

## 에스컬레이션
- 설비 장애/센서 장애 가능 → 관리자 현장 조치 필요
