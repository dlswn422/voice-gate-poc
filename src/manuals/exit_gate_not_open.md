# 출구 차단기 안 열림 (출차 실패/출구 차단봉 미상승) - EXIT_FLOW_ISSUE

## 대표 키워드
출구, 출차, 차단기, 차단봉, 안 올라가요, 정산 완료, 결제 완료, 출차기록, LPR, 영수증

## 증상(사용자 발화 예)
- "출구에서 차단봉이 안 올라가요"
- "결제했는데 출차가 안 돼요"
- "정산 완료 떴는데 막혀요"

## 확인 질문(필요 시 1개)
- "방금 결제(정산) 완료 화면을 보셨나요?"

## 조치 순서(우선순위)
1) 출구 정지선/루프코일 위에 정확히 정차(앞뒤 이동 후 재감지)
2) 결제 완료 여부 확인: 키오스크 결제/모바일 결제/사전정산 여부 확인
3) 출구 단말에서 차량번호 인식 상태 확인(번호판 오염/가림 시 닦고 재시도)
4) 결제했는데도 계속 실패하면 출구 호출 버튼으로 관제 연결(결제/입차기록 확인 요청)
5) 혼잡 시 안전 확보 후 관제 지시에 따라 수동 처리
