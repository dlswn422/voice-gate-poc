# 출차가 안 됨(출구 차단기 안 열림)

## 증상
- "나가야 하는데 안 열려요", "출차 차단기가 안 올라가요", "결제했는데 출차가 안돼요"

## 조치 순서(우선순위)
1) 결제 완료 여부 확인: 결제 완료 화면/영수증 여부
2) 차량 위치 확인: 센서 앞 정위치(너무 멀면 출차 인식 실패)
3) 결제했는데도 안 열리면: 결제 반영 지연 가능 → 잠시 후 재시도 안내
4) 계속 실패하면: 출구 호출 버튼/안내번호로 관리자 호출 안내
5) 안전: 차단봉을 억지로 밀지 않도록 안내

## 추가 질문(필요 시 1개만)
- "결제는 완료 화면이 떴나요, 아니면 결제 자체가 진행이 안 됐나요?"

## 에스컬레이션
- 결제 반영/출차 설비 연동 장애 가능 → 관리자 확인 필요
