# 출구 차단기 안 열림 (출차 실패/출구 차단봉 미상승) - EXIT_FLOW_ISSUE

## 대표 키워드
- 출구 차단기 안 열림
- 출차 안 돼요
- 정산 완료
- 결제 완료
- 입차 기록 없음
- 번호판 인식
## 필수 슬롯 (required_slots)
- symptom: 어떤 현상인지(정산완료인데 안 열림/입차기록 없음/번호판 인식 문제 등)
- where: 출구 차단기/출구 정산기/차량 인식 구간 중 어디인지
## 추가 슬롯 (optional_slots)
- error_message: 화면 오류 문구(예: '입차 기록 없음')
- attempted: 앞뒤 이동/재정산/다른 차로 시도 여부
## 첫 질문 (ASK_1)
출차에서 어디에서 막히시나요? (차단기/정산/차량 인식) 그리고 화면에 '입차 기록 없음' 같은 문구가 보이나요?
## 슬롯 추출 가이드
- '정산 완료 떴는데 안 열려요' → symptom=정산완료인데 차단기 미동작
- '입차 기록이 없대요' → symptom=입차기록 없음, error_message=입차 기록 없음
- '번호판이 다르게 떠요' → symptom=번호판 불일치
## 진단 분기 (결정 규칙)
1) error_message에 '입차 기록 없음' 또는 번호판 불일치가 의심되면 → lpr_mismatch_or_no_entry_record 참고
2) 결제/정산 자체가 불확실하면 → 결제 위치(where) 확인 후 재정산 유도
3) 정산 완료인데도 안 열리면 → 정지선/루프코일 재감지 + 관리자 호출 기준
## 조치 순서 (우선순위)
1) 출구 정지선/루프코일 위에 정확히 정차(앞뒤 이동 후 재감지)
2) 결제 완료 여부 확인(키오스크/모바일/사전정산/출구정산)
3) 출구 단말의 차량번호 인식 상태 확인(번호판 오염/가림 시 닦고 재시도)
4) 화면에 '입차 기록 없음' 또는 번호판 불일치가 뜨면 관리자/관제 호출(수동 확인 필요)
5) 계속 실패하면 관리자 호출
## 해결 안내 문장 템플릿 (SOLVE_TEMPLATE)
- 차량을 정지선(루프코일) 위에 맞추고 앞뒤로 조금 이동한 뒤 다시 시도해 주세요.
- 결제가 완료됐는지(키오스크/모바일/출구정산) 확인 후 필요하면 재정산해 주세요.
- 화면에 '입차 기록 없음'이 뜨면 번호판 인식/입차 기록 확인이 필요해요. 관리자를 호출해 주세요.
- 다른 문제가 더 있으신가요?
