# FAILSAFE + 세션 종료(DONE) 가이드

## 목적
- 사용자가 애매하게 말하거나(모호/비관련/욕설), 해결/종료 의사를 표현할 때
  상담이 무한루프에 빠지지 않고 안전하게 마무리되도록 한다.

## 종료(DONE) 트리거 예시
- "됐어요", "해결됐어요", "괜찮아요", "그만할게요", "종료", "끝", "감사합니다", "안녕"

## FAILSAFE 트리거 예시
- 사용자가 "무슨 말인지 모르겠어요", "다시 말해줘", "엉뚱한 답이에요"
- 메뉴얼과 전혀 무관한 주제, 의미 없는 반복, 욕설/비방

## 조치 순서(우선순위)
1) (DONE) 종료 의사면: 감사/배웅 멘트 + 상담 종료 처리
2) (FAILSAFE) 이해 실패면: 한 문장으로 상황 요약 + 1개 질문만 요청
3) 2회 이상 FAILSAFE 반복되면: 관리자 호출/안내번호 안내

## 권장 배웅 문구 예시
- "네, 해결되셨다니 다행입니다. 이용해 주셔서 감사합니다. 안녕히 가세요."
