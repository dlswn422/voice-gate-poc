# 차단기(차단봉) 안 열림 - 공통(입차/출차 구분 전)

## 증상(사용자 발화 예시)
- "차단기가 안 열려요"
- "차단봉이 안 올라가요"
- "문이 안 열려요", "열리질 않아요"
- "통과해야 하는데 막혀요"
- "차가 인식이 안 돼요"
- "그냥 멈춰있어요", "반응이 없어요"

## 상담 목표(핵심)
- 먼저 **입차인지 출차인지**를 1문장으로 확인한다.
- 차량이 차단기 앞에 정상 정지했는지(센서 감지) 확인한다.
- 결제/등록/인식 문제인지, 물리/장비 문제인지 분기한다.
- 즉시 해결이 어렵다면 **현장 도움 요청(콜 버튼/관리자 호출)**로 안전하게 에스컬레이션한다.

## 1) 가장 먼저 물어볼 1가지(필수 질문)
- "입차(들어갈 때)인가요, 출차(나갈 때)인가요?"

## 2) 공통 점검(우선순위)
1) 차량 위치 확인
- 차단기 바로 앞(정지선/루프코일 센서)까지 충분히 접근했는지 확인
- 너무 멀거나 너무 가까우면 감지가 안 될 수 있으니, 천천히 30~50cm 정도 다시 맞춰 정지

2) 주변 안전 확인
- 보행자/차량이 차단봉 아래에 있으면 절대 강제 개방 시도 금지
- 안전이 확보된 상태에서만 다음 조치 안내

3) 안내 표시/알림 확인(가능하면)
- 키오스크/표시등에 안내 문구(결제 필요, 등록 필요, 인식 실패 등)가 뜨는지 확인
- 경고음/빨간불/오류코드가 있는지 확인

## 3) 분기 안내(입차/출차 확정 전 임시 분기)
- "차량번호 인식(번호판 인식)이 안 된 것 같아요" → 차량 인식/등록 문제 가능
- "결제하고 나가려는데 안 열려요" → 결제/정산 문제 가능
- "아무 표시도 없고 반응이 없어요" → 장비/네트워크/센서 문제 가능

## 4) 즉시 안내 문장(사용자에게 자연스럽게)
- "우선 입차인지 출차인지 확인해 주실래요? 그리고 차량이 차단기 앞 정지선까지 충분히 들어가 있는지도 같이 확인 부탁드립니다."

## 5) 에스컬레이션(해결 불가 시)
- 2~3번 조치 후에도 그대로면,
- "출구(또는 입구) 옆 호출 버튼이 있으면 눌러서 관리자 도움을 요청해 주세요."
- "현장 안내번호가 보이면 안내번호로 연락해 주세요."

## 금지/주의
- 차단봉을 손으로 밀거나 강제로 올리도록 유도하지 않는다.
- 차량을 급가속/급진입하도록 안내하지 않는다(사고 위험).
