# 키오스크 UI / 디바이스 장애 안내 - KIOSK_UI_DEVICE_ISSUE

### CASE K-01
**조건**
- symptom: 전원이 안 켜져요
- error_message: 없음

**안내 문장**
키오스크 전원이 켜지지 않는 경우, 일시적인 전원 공급 문제이거나 시스템 지연일 수 있습니다. 잠시 후 다시 확인해 주세요. 계속 전원이 켜지지 않으면 관리실에 문의해 도움을 받아주세요.

### CASE K-02
**조건**
- symptom: 조작(터치)이 안 돼요
- error_message: 없음

**안내 문장**
화면은 보이지만 터치 조작이 되지 않는 경우, 일시적인 시스템 오류일 수 있습니다. 잠시 후 다시 시도해 주세요. 계속 조작이 되지 않으면 관리실에 문의해 주세요.

### CASE K-03
**조건**
- symptom: 조작(터치)이 안 돼요
- error_message: 기기 오류

**안내 문장**
터치 패널 장비 오류로 입력이 정상적으로 인식되지 않을 수 있습니다. 추가 조작 없이 관리실에 문의해 도움을 받아주세요.

### CASE K-04
**조건**
- symptom: 조작(터치)이 안 돼요
- error_message: 통신/서버 오류

**안내 문장**
통신 또는 서버 오류로 화면 조작이 정상적으로 처리되지 않을 수 있습니다. 잠시 후 다시 시도해 보시고, 문제가 반복되면 관리실에 문의해 주세요.
