# 키오스크/정산기 화면·버튼 문제 (먹통/터치/프린터) - KIOSK_UI_DEVICE_ISSUE

## 대표 키워드
- 화면 멈춤
- 먹통
- 터치 안됨
- 버튼 무반응
- 영수증 안 나옴
- 프린터
- 재시작
- 오류 화면
## 필수 슬롯 (required_slots)
- symptom: 어떤 UI/장치 문제인지(터치/버튼/화면 멈춤/영수증 출력 등)
- where: 어느 단말인지(키오스크/정산기/출구 단말)
## 추가 슬롯 (optional_slots)
- error_message: 화면 문구
- attempted: 다른 단말로 시도/재시도 여부
## 첫 질문 (ASK_1)
화면이 멈춘 건가요, 아니면 터치/버튼만 반응이 없나요? (키오스크/출구 정산기 중 어디인가요?)
## 슬롯 추출 가이드
- '먹통/멈췄' → symptom=화면 멈춤
- '버튼이 안 눌려요/터치 안돼요' → symptom=버튼/터치 무반응
- '영수증이 안 나와요' → symptom=영수증 출력 실패
## 조치 순서 (우선순위)
1) 일시적 멈춤이면 10~20초 대기 후 다시 입력/재시도
2) 가능하면 다른 키오스크/다른 차로 단말을 이용 안내
3) 영수증 출력만 실패한 경우, 요금/정산 완료 여부는 화면에서 먼저 확인 안내
4) 계속 무반응/먹통이면 관리자 호출(단말 점검 필요)
## 해결 안내 문장 템플릿 (SOLVE_TEMPLATE)
- 단말이 일시적으로 멈출 수 있어요. 10~20초 기다린 뒤 다시 시도해 주세요. 가능하면 다른 단말로도 시도해 주세요.
- 터치/버튼이 계속 무반응이면 단말 점검이 필요합니다. 관리자를 호출해 주세요.
- 다른 문제가 더 있으신가요?
