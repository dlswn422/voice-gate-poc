# 차량번호 인식 오류 / 입차 기록 없음

## 증상
- "차번호가 다르게 떠요", "입차 기록이 없대요", "내 차가 목록에 없어요"

## 조치 순서(우선순위)
1) 화면에 표시된 차량번호가 맞는지 확인(오인식 가능)
2) 번호판 오염/가림 여부 확인 후 재인식 유도(가능한 경우)
3) 입차 직후라면 반영 지연 가능 → 잠시 후 재시도
4) 계속 오류면 관리자 호출(수기 확인/입차기록 확인 필요)
5) 출차가 급하면 출구 호출 버튼/안내번호로 도움 요청

## 추가 질문(필요 시 1개만)
- "화면에 표시된 차량번호가 실제 차량번호와 어떻게 다른가요?"
