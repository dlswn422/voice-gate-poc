# 번호판 인식 오류 / 입차기록 없음 (LPR 불일치) - LPR_MISMATCH

## 대표 키워드
번호판 인식, LPR, 입차기록 없음, 차량번호 불일치, 인식 실패, 오염, 가림, 재인식

## 증상(사용자 발화 예)
- "입차 기록이 없대요"
- "차량번호가 다르게 뜨는 것 같아요"
- "번호판 인식이 안 됐나봐요"

## 확인 질문(필요 시 1개)
- "정산기 화면에 표시된 차량번호가 실제 번호와 다른가요?"

## 조치 순서(우선순위)
1) 차량번호 확인: 화면 표시 번호와 실제 번호 비교
2) 번호판 상태 점검: 오염/가림/훼손 여부 확인 후 닦고 재시도
3) 다른 차로/다른 키오스크에서 재조회(재인식 유도)
4) 입차기록이 없으면 관제/관리자 호출(수동 입차 등록/기록 확인 요청)
5) 출차 지연 시 안전 확보 후 관제 지시에 따른 수동 처리

## 에스컬레이션(관리자/관제 안내)
- 특정 구간에서 반복이면 LPR 카메라 각도/조명/오염 점검 필요
