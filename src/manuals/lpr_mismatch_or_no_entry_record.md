# 번호판 인식 오류 / 입차기록 없음 (LPR 불일치) - LPR_MISMATCH

## 대표 키워드
- 입차 기록 없음
- 번호판 인식 실패
- 차량번호 불일치
- LPR
- 오염
- 가림
- 재인식
## 필수 슬롯 (required_slots)
- symptom: 입차기록 없음/번호판 불일치/인식 실패 중 무엇인지
- where: 출구/키오스크/입구 중 어느 구간에서 발생했는지
## 추가 슬롯 (optional_slots)
- error_message: 화면 문구(예: '입차 기록 없음')
- attempted: 번호판 닦기/다른 차로 재조회 여부
## 첫 질문 (ASK_1)
화면에 표시된 차량번호가 실제 번호와 다른가요, 아니면 '입차 기록 없음'이라고 나오나요?
## 슬롯 추출 가이드
- '입차 기록 없음' → symptom=입차기록 없음, error_message=입차 기록 없음
- '번호판이 다르게 떠요' → symptom=번호판 불일치
- '인식이 안 됐나봐요' → symptom=인식 실패
## 조치 순서 (우선순위)
1) 화면 표시 차량번호와 실제 차량번호를 비교해 확인
2) 번호판 오염/가림/훼손 여부 확인 후 닦고 재시도(야간 반사/흙/눈/커버 등)
3) 가능하면 다른 차로/다른 키오스크에서 재조회(재인식 유도)
4) 입차 기록이 없거나 불일치가 계속되면 관제/관리자 호출(수동 입차 등록/기록 확인 요청)
## 해결 안내 문장 템플릿 (SOLVE_TEMPLATE)
- 번호판 인식 문제는 오염/가림 때문에 발생할 수 있어요. 번호판을 닦고 다시 인식되도록 시도해 주세요.
- 계속 '입차 기록 없음'이 뜨면 시스템에서 기록 확인이 필요합니다. 관리자를 호출해 주세요.
- 다른 문제가 더 있으신가요?
## 에스컬레이션 조건
- 번호판을 닦고 재시도해도 동일
- 여러 차로/키오스크에서 동일 증상
- 시스템에 입차 기록이 실제로 존재하지 않는 것으로 보일 때
