# 모바일/QR 사전정산 문제 (앱/QR/번호 입력) - MOBILE_PAYMENT_QR_ISSUE

## 대표 키워드
- 모바일 결제
- 사전정산
- QR
- 앱
- 인증
- 번호 입력
- 결제 완료
- 반영 안 됨
## 필수 슬롯 (required_slots)
- symptom: 어떤 문제인지(QR 인식/결제 실패/결제 완료인데 반영 안 됨)
- where: 모바일/QR 화면인지, 출구/키오스크에서 확인 중인지
## 추가 슬롯 (optional_slots)
- error_message: 앱/웹 화면 오류 문구
- attempted: 새로고침/재로그인/재결제 여부
## 첫 질문 (ASK_1)
모바일에서 결제가 실패한 건가요, 아니면 결제는 완료됐는데 출구/키오스크에서 반영이 안 되나요?
## 슬롯 추출 가이드
- '결제 자체가 안돼요' → symptom=모바일 결제 실패
- '결제 완료인데 출차가 안돼요' → symptom=정산 반영 문제
- 'QR이 인식이 안돼요' → symptom=QR 인식 실패
## 조치 순서 (우선순위)
1) 모바일 결제 실패면 네트워크 상태 확인 후 앱 새로고침/재시도
2) 결제 완료인데 반영 안 되면 결제 완료 화면/영수증을 확인하고 10~20초 후 재조회
3) QR 인식 실패면 화면 밝기 올리고 QR을 가까이 대어 재시도
4) 계속 실패하면 키오스크/출구 정산기로 대체 안내 또는 관리자 호출
## 해결 안내 문장 템플릿 (SOLVE_TEMPLATE)
- 모바일 결제는 네트워크 상태에 따라 실패할 수 있어요. 새로고침 후 재시도해 주세요.
- 결제 완료인데 반영이 안 되면 10~20초 후 재조회해 보시고, 계속되면 관리자를 호출해 주세요.
- 다른 문제가 더 있으신가요?
