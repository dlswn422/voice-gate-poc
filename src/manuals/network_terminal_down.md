# 네트워크/단말 장애 (키오스크/정산기/입출차기 통신 문제) - NETWORK_DOWN

## 대표 키워드
- 통신 오류
- 서버 연결 실패
- 오프라인
- 네트워크 장애
- 화면 멈춤
- 먹통
- 단말 장애
## 필수 슬롯 (required_slots)
- symptom: 어떤 오류인지(통신/서버/오프라인/화면 멈춤 등)
- where: 어떤 단말인지(키오스크/정산기/입구/출구)
## 추가 슬롯 (optional_slots)
- error_message: 화면 문구(통신/서버/오프라인 등)
- attempted: 다른 단말/차로로 시도했는지
## 첫 질문 (ASK_1)
화면에 '통신/연결/서버/오프라인' 같은 문구가 보이나요? 그리고 어느 단말(키오스크/출구/입구)에서 발생했나요?
## 슬롯 추출 가이드
- '서버 연결 오류/통신 오류' → symptom=통신/서버 오류, error_message 기록
- '먹통/멈춤/반응없음' → symptom=화면 멈춤/단말 먹통
- where: 키오스크/정산기/입구/출구 구분
## 조치 순서 (우선순위)
1) 단말 상태 확인(화면 멈춤/오류 문구/재시도 버튼 유무)
2) 가능하면 다른 키오스크/다른 차로로 시도 안내
3) 일시적 장애 가능성이 있으면 10~20초 후 재시도
4) 계속 장애면 관리자/관제 호출(단말/네트워크 점검 요청)
## 해결 안내 문장 템플릿 (SOLVE_TEMPLATE)
- 통신/서버 오류가 반복되면 단말 또는 네트워크 장애일 수 있어요. 가능하면 다른 단말/차로로 시도해 주세요.
- 계속 동일하면 관리자를 호출해 점검을 요청해 주세요.
- 다른 문제가 더 있으신가요?
## 에스컬레이션 조건
- 여러 단말/차로에서 동시에 동일 오류
- 20초 이상 지속적으로 '오프라인/서버 연결 실패' 표시
- 화면 멈춤이 반복
