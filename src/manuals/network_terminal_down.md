# 네트워크/단말 장애 (키오스크/정산기/입출차기 통신 문제) - NETWORK_DOWN

## 대표 키워드
네트워크, 통신 장애, 단말 장애, 오프라인, 연결 실패, 서버 오류, 재부팅, 결제 단말

## 증상(사용자 발화 예)
- "키오스크가 먹통이에요"
- "서버 연결 오류가 떠요"
- "결제가 안 되고 화면이 멈춰요"

## 확인 질문(필요 시 1개)
- "화면에 '연결/통신/서버' 관련 오류 문구가 있나요?"

## 조치 순서(우선순위)
1) 단말 상태 확인: 화면 멈춤/오류 문구/재시도 버튼 유무 확인
2) 동일 단말 반복 여부 확인: 다른 키오스크/다른 차로 사용 가능하면 안내
3) 일시적 장애면 10~20초 후 재시도(연결 복구 기다림)
4) 계속 장애면 관제/관리자 호출(단말/네트워크 점검 요청)
5) 출차 지연 시 안전 확보 후 관제 지시에 따른 수동 처리

## 에스컬레이션(관리자/관제 안내)
- 다수 단말 동시 장애면 네트워크/서버 장애 가능성 높음
