# 네트워크/단말 장애 안내 - NETWORK_TERMINAL_DOWN

### CASE NTD-01
**조건**
- symptom: 네트워크 연결이 안 돼요
- error_message: 없음

**안내 문장**
네트워크 연결이 일시적으로 불안정할 수 있습니다. 잠시 후 다시 시도해 주세요. 문제가 계속되면 관리실에 문의해 도움을 받아주세요.

### CASE NTD-02
**조건**
- symptom: 네트워크 연결이 안 돼요
- error_message: 통신 오류

**안내 문장**
통신 오류로 단말이 네트워크에 정상적으로 연결되지 않을 수 있습니다. 잠시 후 다시 시도해 보시고, 문제가 반복되면 관리실에 문의해 주세요.

### CASE NTD-03
**조건**
- symptom: 네트워크 연결이 안 돼요
- error_message: 오프라인 상태

**안내 문장**
단말이 현재 오프라인 상태로 확인됩니다. 네트워크 복구 후 자동으로 정상 동작할 예정이니 잠시만 기다려 주세요. 장시간 지속되면 관리실에 문의해 주세요.

### CASE NTD-04
**조건**
- symptom: 서버에 연결할 수 없대요
- error_message: 없음

**안내 문장**
서버 연결이 일시적으로 지연되고 있을 수 있습니다. 잠시 후 다시 시도해 주세요. 문제가 계속되면 관리실에 문의해 주세요.

### CASE NTD-05
**조건**
- symptom: 서버에 연결할 수 없대요
- error_message: 서버 응답 없음

**안내 문장**
서버 응답이 없어 처리가 지연되고 있습니다. 잠시 후 다시 시도해 보시고, 동일한 문제가 반복되면 관리실에 문의해 주세요.

### CASE NTD-06
**조건**
- symptom: 계속 로딩만 돼요
- error_message: 없음

**안내 문장**
네트워크 또는 서버 응답 지연으로 화면 전환이 늦어질 수 있습니다. 잠시만 기다려 주세요. 로딩 상태가 오래 지속되면 관리실에 문의해 주세요.

### CASE NTD-07
**조건**
- symptom: 계속 로딩만 돼요
- error_message: 서버 응답 없음

**안내 문장**
서버 응답 지연으로 처리가 완료되지 않고 있습니다. 잠시 후 다시 시도해 보시고, 문제가 계속되면 관리실에 문의해 주세요.

### CASE NTD-08
**조건**
- symptom: 계속 로딩만 돼요
- error_message: 통신 오류

**안내 문장**
통신 오류로 요청이 정상 처리되지 않아 로딩 상태가 지속될 수 있습니다. 잠시 후 다시 시도해 보시고, 문제가 반복되면 관리실에 문의해 주세요.
