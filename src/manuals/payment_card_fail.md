# 카드 결제 실패 (정산기/키오스크/출구정산) - PAYMENT_ISSUE

## 대표 키워드
카드 결제, 승인 실패, 결제 오류, 정산기, 키오스크, 출구정산, IC칩, 마그네틱, 다른 카드

## 증상(사용자 발화 예)
- "카드 결제가 안 돼요"
- "승인이 안 돼요 / 결제 오류가 떠요"
- "정산기에서 카드가 안 먹어요"

## 확인 질문(필요 시 1개)
- "지금 정산기(키오스크) 화면에 '승인 실패/오류'가 표시되나요?"

## 조치 순서(우선순위)
1) 카드 방향/삽입 상태 확인 (IC칩/마그네틱 방향이 맞는지)
2) IC칩(금속면) 오염 여부 확인 → 닦은 뒤 재시도
3) 다른 카드로 결제 시도 (가능하면 다른 브랜드/다른 결제수단)
4) 계속 실패하면 정산기 단말/네트워크 상태 점검 필요 → 관리자(관제) 안내
5) 출차 지연이면 출구 호출 버튼/안내번호로 도움 요청

## 에스컬레이션(관리자/관제 안내)
- 동일 정산기에서 다수 결제 실패면 단말/네트워크 장애 가능성 ↑
- 출구 혼잡 발생 시 수동 처리(관제 지시에 따름)
