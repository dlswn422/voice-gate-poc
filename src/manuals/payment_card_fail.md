# 카드 결제 실패(정산기/키오스크)

## 증상
- "카드 결제가 안돼요", "승인이 안돼요", "결제 오류", "카드가 먹통이에요"

## 조치 순서(우선순위)
1) 카드 방향/삽입 상태 확인(IC칩/마그네틱 방향, 끝까지 삽입)
2) IC칩(금속면) 오염 시 닦은 뒤 재시도
3) 다른 카드로 결제 시도(가능하면 다른 브랜드/다른 결제수단)
4) 계속 실패하면 단말/네트워크 상태 확인(관리자 안내)
5) 출차 지연 시 출구 호출 버튼/안내번호로 도움 요청

## 추가 질문(필요 시 1개만)
- "카드를 넣었을 때 '승인 실패' 문구가 나오나요, 아니면 카드 인식 자체가 안 되나요?"

## 에스컬레이션(관리자/현장 안내)
- 동일 증상이 반복되면 단말 장애/네트워크 장애 가능 → 관리자 확인 요청
