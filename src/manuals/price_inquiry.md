# 주차 요금/가격 문의

## 증상
- "요금 얼마예요?", "주차비가 어떻게 돼요?", "몇 시간에 얼마예요?"

## 조치 순서(우선순위)
1) 사용자가 원하는 정보 유형 확인: "현재까지 요금" vs "요금표/기준"
2) 현재까지 요금 문의라면: 입차 시간/차량번호/주차권(있으면) 확인 안내
3) 요금표/기준 문의라면: 기본요금/추가요금/일최대요금/무료시간 여부를 안내
4) 할인/무료 적용 가능성이 있으면 "할인/무료 적용" 메뉴얼로 연결

## 추가 질문(필요 시 1개만)
- "현재 결제하려는 건가요, 아니면 요금 기준(요금표)을 문의하시는 건가요?"

## 에스컬레이션
- 요금이 예상과 다르면 입차기록/할인적용 확인 필요 → 관리자 확인 요청
