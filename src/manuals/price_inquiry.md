# 주차 요금 문의 (요금/정산/무료시간/할인) - PRICE_INQUIRY

## 대표 키워드
요금, 주차비, 얼마, 정산, 무료시간, 할인, 회차, 시간당, 일최대, 영수증

## 증상(사용자 발화 예)
- "주차비 얼마예요?"
- "지금 정산하면 얼마 나와요?"
- "무료시간/할인 적용됐나요?"

## 확인 질문(필요 시 1개)
- "현재 입차 시간이 언제인지(대략) 기억하시나요?"

## 안내 순서(우선순위)
1) 요금은 '입차시간~정산시간' 기준으로 계산됨을 안내
2) 무료시간/할인은 조건(방문처/영수증/등록 여부)에 따라 적용됨을 안내
3) 키오스크/정산기에서 차량번호 확인 후 요금 확인 유도
4) 할인 미적용이면 방문처(매장/안내데스크)에서 할인 등록 필요 안내
5) 그래도 불일치면 관리자(관제) 확인 필요 안내

## 에스컬레이션(관리자/관제 안내)
- 요금이 비정상적으로 높게 나오거나 입차 기록이 없다고 나오는 경우 → LPR/입차기록 문제 가능
