# 방문등록 실패

## 증상
- "방문등록이 안돼요", "등록 버튼이 안 눌려요", "등록했는데 적용이 안돼요"

## 조치 순서(우선순위)
1) 입력한 차량번호 형식/띄어쓰기 확인(예: 12가3456)
2) 등록 대상(해당 주차장/해당 시간대)인지 확인
3) 등록 완료 메시지/화면 확인 후, 정산 화면에서 반영 여부 확인
4) 계속 실패하면: 시스템 지연/권한/네트워크 문제 가능 → 관리자 안내
5) 출차 급하면 출구 호출 버튼/안내번호로 도움 요청

## 추가 질문(필요 시 1개만)
- "차량번호를 입력하면 오류 메시지가 뜨나요, 아니면 등록은 되는데 적용이 안 되나요?"

## 에스컬레이션
- 등록 시스템/서버 장애 또는 정책 제한 가능 → 관리자 확인 필요
