# 등록 실패 / 정기권 오류 (REGISTRATION)

## 대표 키워드
- 정기권
- 정기 차량
- 등록된 차
- 문이 안 열려요

## 필수 슬롯
- symptom: 정기권 인식 안됨/등록 확인

### CASE R-01
**조건**
- [DB 확인] 정기권 유무: **있음(O)** (VALID)
**안내 문장**
정기권이 정상적으로 확인되었습니다.
차단기를 열어드리겠습니다. 안녕히 가세요.

### CASE R-02
**조건**
- [DB 확인] 정기권 유무: **없음(X)** (INVALID)
**안내 문장**
현재 차량은 정기권 정보가 확인되지 않습니다.
관리자를 호출해 드리겠습니다. 잠시만 기다려 주세요.
