# 방문자/차량 등록 실패(방문등록/번호판 등록/사전등록 오류)

## 증상(사용자 발화 예시)
- "방문 등록이 안돼요"
- "차량번호 등록했는데 오류가 나요"
- "등록했는데 입차가 안돼요"
- "방문 예약했는데 인식이 안돼요"
- "등록 버튼 눌러도 다음으로 안 넘어가요"
- "번호판 입력했는데 틀렸다고 떠요"

## 상담 목표(핵심)
- 등록은 보통 "입력 오류(번호판/전화번호/인증)" 또는 "사전등록 미반영" 또는 "네트워크/시스템 오류" 3가지로 많이 갈린다.
- 사용자가 빠르게 해결하려면: **입력값 확인 → 등록 완료 여부 확인 → 반영 지연 안내 → 에스컬레이션** 순서

## 1) 먼저 확인할 질문(1개만)
- "등록하려는 게 방문자 등록(방문 예약)인가요, 차량번호(번호판) 등록인가요?"

## 2) 조치 순서(우선순위)
1) 입력값 형식 확인(가장 흔한 원인)
- 번호판: 띄어쓰기 제거, 한글/숫자 순서 확인(예: '12가3456')
- 전화번호/인증번호: 자리수/오타 확인
- 문구/경고창이 있으면 그대로 읽어달라고 요청(가능하면)

2) 등록 완료/저장 여부 확인
- "등록 완료" 메시지 또는 완료 화면으로 넘어갔는지 확인
- 완료를 못 봤다면, 다시 시도(단 동일 입력 반복으로 잠길 수 있으면 1~2회만)

3) 반영 지연(동기화) 안내
- "등록 직후 시스템 반영까지 몇 초~1분 정도 걸릴 수 있어요."
- 입차/출차 시도 전에 잠시 대기 후 재시도 안내

4) 사전등록/예약 정보 불일치 가능성
- 방문 시간/차량번호가 예약 정보와 다른 경우
- 다른 차량/다른 번호판으로 등록한 경우
- 예약이 만료되었거나 날짜가 다른 경우

5) 계속 실패 시 에스컬레이션
- 현장 호출 버튼/관리자 안내로 등록 확인 요청
- 관리자가 백오피스에서 직접 등록/수정할 수 있는지 확인

## 3) 예시 답변(자연스럽게)
- "방문자 등록인지 차량번호 등록인지 먼저 확인해 주실래요? 번호판이라면 띄어쓰기 없이 '12가3456' 형태로 입력했는지 확인해 주세요. 등록 완료 메시지가 떴는데도 안 되면, 반영까지 30초 정도 기다렸다가 다시 시도해 보세요."

## 주의/팁
- 사용자가 흥분/불만(complaint)일 때는 해결 질문 1개만 던지고, 바로 다음 행동(관리자 호출 등)을 함께 제시한다.
